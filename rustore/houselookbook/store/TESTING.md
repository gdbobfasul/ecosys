# Pupikes Chat — Testing notes (RUStore)

## Какво прави приложението
WebView обвивка, която зарежда продукционния чат `https://my.girl.place`.

## Тестови стъпки
1. Инсталирай APK на устройство/емулатор с интернет.
2. Стартирай → трябва да се зареди живият чат (страницата за вход/чат).
3. Тествай вход с тестов акаунт (виж екипа за тестови данни).
4. Изпрати/получи съобщение → проверка, че реалният сървър отговаря.

## Офлайн поведение
1. Изключи интернет (самолетен режим).
2. Стартирай приложението → показва екран „Няма връзка със сървъра" + бутон
   „Опитай пак".
3. Включи интернет → натисни „Опитай пак" → чатът се зарежда.

## Без следните функции (умишлено)
- Без вградени плащания (IAP).
- Без GMS/HMS/Firebase, без push през тези услуги в обвивката.
- Без достъп до контакти, без проследяване от обвивката.
