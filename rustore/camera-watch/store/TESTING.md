# MotionHawk — тестов сценарий (RUStore ревю)

App ID: `com.kcy.camerawatch.rustore`

## Без акаунт
Приложението не изисква регистрация/вход. Отваряш и ползваш.

## Стъпки
1. **Старт → Активирай безплатно.** Екран с честно обяснение; бутон активира (без плащане).
2. **Разрешения.**
   - „Разреши камера“ → системен диалог → дай достъп.
   - „Разреши нотификации“ → системен диалог → дай достъп (по избор; без него събитията пак се виждат в журнала).
3. **Настройки.**
   - Източник: остави „Камера на телефона“.
   - Чувствителност: средна.
   - Маркирай „Човек / нарушител“ и „Животни“.
   - Запази → към таблото.
4. **Табло → „Постави на пост (Arm)“.**
   - Появява се живо видео от задната камера.
   - Помахай с ръка / мини пред камерата → статусът става „Движение засечено…“, после „ДЕТЕКЦИЯ: човек / нарушител“.
   - Идва **локална нотификация** и нов ред в **журнала** със снимка.
5. **„Свали от пост (Disarm)“** спира камерата.

## „Друга камера“ (по избор)
- В Настройки → Източник → „Друга камера“ → въведи URL на browser-playable поток (MJPEG/HLS).
- Ако потокът не е възпроизводим в браузър или няма CORS, приложението го казва честно (RTSP не се поддържа директно).

## Среда без камера (емулатор/headless)
- Приложението НЕ се срива: показва ясно съобщение „не открих камера / достъпът е отказан“.

## Очаквано
- Без сривове, без вход, без плащане, без искане на контакти/локация.
- Първото разпознаване сваля AI модела веднъж (кратко изчакване), после е бързо.
