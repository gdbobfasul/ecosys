# RUStore — затворено тестване / тестова среда

## Тестова среда (без магазин)
Основната тестова среда е **браузърът**:
```bash
npm install && npm run dev
```
Всеки офлайн инструмент е напълно функционален в браузъра. APK за устройство се
билдва отделно (виж README → Android APK; локалната машина няма Android SDK).

## Затворено тестване в RUStore
RUStore поддържа разпространение чрез:
1. **Тестова версия (черновик)** — качваш APK/AAB в RUStore Console като чернова и
   я инсталираш на собствени тестови устройства преди публикуване.
2. **Линк за тестери** — споделяш достъп до приложението с ограничен кръг тестери
   през конзолата (раздел „Тестирование").

### Стъпки
1. Регистрирай разработчик в https://www.rustore.ru/ (RUStore Console).
2. Създай ново приложение → App ID `com.pupikes.servicestoolkit.rustore`.
3. Качи подписан APK/AAB (debug е достатъчен за вътрешно тестване; за публикуване —
   release с твой keystore — виж submission-checklist.md).
4. Попълни листинга (listing-notes.md), икона и екранни снимки.
5. Изпрати към „Тестирование" и добави тестери по e-mail/линк.
6. След потвърждение от тестерите → подай за модерация/публикуване.

## Контролен лист за тестера (ръчно на устройство)
- [ ] Началният екран показва всички карти; търсачката филтрира.
- [ ] QR: генерира код, сваля PNG; разчита от снимка; сканира с камера.
- [ ] Парола: четирите метода дават резултат; индикаторът за сила работи; копира.
- [ ] Калкулатор: 4-те таба дават коректни числа.
- [ ] Текст: броячите се обновяват; форматиранията и Base64 работят.
- [ ] Снимки: компресира и сваля; показва спестения размер.
- [ ] PDF: слива, разделя, добавя воден знак; визуален подпис се поставя.
- [ ] Свиване на PDF: намалява размера и сваля файла.
- [ ] Онлайн екраните показват ясна бележка „изисква онлайн услуга".
- [ ] Приложението работи в самолетен режим (офлайн инструментите).
