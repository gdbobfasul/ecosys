# Безопасност на данните — Pupikes Site Monitor (RUStore)

## Събиране на данни: НИКАКВИ
Приложението не събира, не изпраща и не споделя лични данни.

| Категория | Събира ли се? | Споделя ли се? |
|---|---|---|
| Лична информация (име, имейл) | Не | Не |
| Контакти | Не | Не |
| Местоположение | Не | Не |
| Идентификатори / реклами | Не | Не |
| Съобщения / файлове | Не | Не |
| Поведение в приложението / анализ | Не | Не |

## Къде се пазят данните
- Мониторите, настройките и „видяното" състояние се пазят САМО на устройството
  (Capacitor Preferences / localStorage). Няма облачен бекенд и няма акаунт.

## Мрежа
- Приложението прави изходящи заявки САМО към източниците, които потребителят сам
  въведе (RSS/Atom или публично JSON API), и по желание към безплатен CORS прокси,
  който потребителят сам зададе. Нищо не е хардкоднато извън демонстрационните пресети.

## Източници
- Само БЕЗПЛАТНИ източници без ключ. Без вградени API ключове, без платени услуги.

## CORS бележка
- Браузърният слой може да получи отказ от произволни сайтове (CORS). Приложението
  показва тези грешки четимо и не претендира, че всеки URL ще работи.

## Известия
- Само ЛОКАЛНИ известия на устройството. Без push, без FCM/HMS, без сървър.

## Без следните
- Без GMS/HMS/Firebase, без реклами, без проследяване, без вход, без контакти.
