# Монитор-робот — RUStore описание

## Кратко
„Наеми" робот, който следи избран от теб източник (RSS/Atom емисия или публично
JSON API) и те известява локално, когато се появи нов запис или съвпадне ключова
дума. Всичко работи на устройството — без акаунт, без облак, без проследяване.

## Пълно описание
Монитор-робот е еднозадачно приложение за наблюдение на източници. Активираш робота,
добавяш един или повече монитори (всеки със собствен източник, правило и честота на
проверка), даваш разрешение за известия — и роботът сам проверява по график и те уведомява.

Какво следи:
- RSS / Atom емисии (новини, блогове, обяви и т.н.)
- публични JSON API без ключ (с настройваем път до списъка и полета id/заглавие)

Правила за съвпадение:
- появил се е нов запис
- съдържа ключова дума (CSV)
- нов запис + ключова дума

Честота: на 15 минути / на час / веднъж дневно.

## Поверителност
- Без акаунти, без вход, без контакти, без местоположение, без реклами.
- Без облачни сървъри и без push: известията са локални.
- Мониторите и „видяното" състояние се пазят само на устройството.
