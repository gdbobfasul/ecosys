# Testing — Pupikes Baby Radar (RUStore)

## Уеб (dev) с истинска уеб камера
1. `npm install`
2. `npm run dev` → отвори показания URL в браузър (Chrome) на лаптоп с камера.
3. Премини: Онбординг → Активирай → Настройки → Разрешения (Разреши камера + известия) → Наблюдение.
4. Натисни **Старт**. Виж живото видео + бейджа за състояние.
5. **Движение/спи/буден:** Стой неподвижно пред камерата ~20 сек → бейджът става „Спи спокойно“.
   Размахай ръка рязко → „Размърдва се“, а устойчивото движение след спокойствие → **„Събуди се“**
   (известие + звук + запис в дневника).
6. **Втори човек:** изчакай „модел: готов ✓“, после нека влезе втори човек в кадъра → сигнал
   „Непознат в стаята“.
7. **Излезе от кадър:** излез изцяло от кадъра за ~6 сек → сигнал „Излезе от кадър“.
8. **Дневник:** виж записите със снимки; „Изчисти“ ги трие.

## Друга камера (URL) и relay
- В Настройки → Камера → „Друга камера (URL)“ въведи browser-playable поток с CORS (HLS/MJPEG/.mp4).
  RTSP няма да работи (документирано — нужен е gateway).
- Relay URL: въведи адрес на твой сървър; при събитие приложението прави best-effort POST.
  Без сървър няма как известие да стигне до друг телефон (документирано честно).

## Android (RUStore build)
- `npm run cap:add` (веднъж), после `npm run cap:sync`, отвори в Android Studio и пусни на устройство.
- Дай разрешения за камера и известия при поискване.
- НЕ изграждаме APK тук (без gradle в това репо).

## Headless / без камера
- Приложението стартира нормално; при липса на камера „Старт“ показва честно съобщение и не блокира.
