Стъпки за Windows:

**Бързо (просто преинсталация):**
1. Control Panel → Programs → деинсталирай Node.js v24
2. Свали Node 22 LTS от https://nodejs.org/en/download (изберѝ "LTS")
3. Инсталирай с default settings
4. Затвори и пусни нов PowerShell (за да обнови PATH-а)
5. Провери: `node --version` → трябва да покаже `v22.x.x`
6. В папката на проекта:
   ```powershell
   rmdir /s /q node_modules
   npm install
   ```
