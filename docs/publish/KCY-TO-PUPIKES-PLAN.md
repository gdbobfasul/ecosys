_Изготвен 2026-07-20. НЕ Е ИЗПЪЛНЕН — чака решения по два въпроса (раздел 2) + „давай".
Марковото досие: `docs/publish/PUPIKES-BRAND.md`._

## 0. Обхват (какво носи името днес)

- **31 приложения × 2 дървета** (`rustore/` + `huawei/`) = 62 папки-издания.
- + десктоп `desktop/selflearning-friend` (.exe), уеб портали `public/`, `private/portals/`,
  Find Best Price (Expo), WhereNoBiz (уеб).
- Груба следа: **~1959 файла** съдържат „Pupikes"; **450** съдържат `com.kcy`; **603** имат

- **31 keystore-а** (по един на приложение) в `keystores/`.

Къде точно живее името (за всяко приложение):
`capacitor.config.json` (`appName`, `appId`) · `android/app/src/main/res/values/strings.xml`
· `src/**` (i18n `app_name` + текстове) · `package.json` / `README.md` · `store/*.md`

`public/promo/kcy-promo.json` · `deploy-scripts/gen-privacy.mjs` (хардкоднат списък) ·
`deploy-scripts/gen-terms.mjs` · политиките в `public/privacy/` · интро/футър слоевете
(„Pupikes" екраните) · метаданните `metadata/*.yaml`.

## 1. Стратегия на именуване (препоръка)

- `Pupikes` → **`Pupikes`** (интро екранът, футър каталогът, промо).
- `Pupikes Toolkit` → **`Pupikes Toolkit`** (и 13-те под-приложения: `Pupikes Toolkit PDF` …).

  и т.н. — навсякъде „Pupikes " → „Pupikes " като представка.
- Преименуваните вече игри (EvadeArena, Godfist Arena, Huntline 3D, Warbird Rush, MotionHawk,
  Rustam picks cucumbers) НЯМАТ „Pupikes" в името → остават както са (може по избор да получат
  „by Pupikes" в описанието).

## 2. ДВЕ решения, които определят целия обхват (чакам отговор)

### Решение A — сменяме ли `applicationId` (`com.kcy.*` → `com.pupikes.*`)?

| | Смяна на пакета | Запазване на пакета |
|---|---|---|
| Видимо име | Pupikes | Pupikes |
| Пакет отдолу | `com.pupikes.<ап>.rustore/.hw` | остава `com.kcy.<ап>.…` (невидим) |
| Вече публикувани | стават **НОВИ** приложения (нов лист, губят инсталации/оценки) | ъпдейт на същото приложение |
| Подпис/keystore | може да се запазят (само appId се мени) | без промяна |
| Чистота на марката | пълна (и технически Pupikes) | само на повърхността |

⚠️ **Важно облекчение:** по вътрешната бележка [[kcy-no-live-data]] цялата екосистема е
**тестова** — няма реални потребители за губене. Ако още НИЩО не е публикувано наживо в
магазините, смяната на пакета е **без цена** и дава напълно чист старт под Pupikes.
→ Ако вече има качени апове с реални инсталации — тогава запазваме пакета.

**Препоръка:** ако не е публикувано наживо → **сменяме пакета** (`com.pupikes.*`), чист старт.

### Решение B — „Pupikes" изчезва напълно, или остава някъде?

- Вариант 1 (препоръчан): „Pupikes" изчезва изцяло; Pupikes е единствената марка.
- Вариант 2: Pupikes е фамилията, но някоя под-линия пази стар етикет (не виждам причина).
