## База данни (SQLite)

4 таблици в `private/eco-3/database/eco3.db`:

- **eco3_uniqueness** — tracker за уникалност (дневно нулиране)
- **eco3_search_history** — история на търсенията
- **eco3_results** — кеширани резултати (за препродажба)
- **eco3_stats** — статистика за админ панел

## Admin IP Whitelist

В `.env`:
```env
ADMIN_ALLOWED_IPS=127.0.0.1,::1,YOUR_REAL_IP
```

Ако IP-то ти не е в списъка → получаваш 403 + подсказка с текущото ти IP.

В development (NODE_ENV != production) — admin е достъпен от всички IP-та.

## Deploy

`server-install.sh` автоматично:
1. Създава ECO-3 SQLite DB от schema.sql
2. Копира .env.example → .env (ако няма)
3. Създава logs/ и database/ директории
4. Задава правилни permissions (kcy-eco3 user)
5. Стартира kcy-eco3 service
6. Показва URL на статус страницата
