# 04 - User Guide (Потребителско ръководство)

## 📱 Как работи Pupikes Chat

Анонимен чат без SMS верификация, без email, само телефон и парола.

---

## 🚀 Регистрация и Вход

### **Първо влизане:**
1. Отвори https://yourdomain.com
2. Въведи телефонен номер
3. Въведи парола
4. Натисни "Влез"

**Логика:**
- Ако НЕ съществуваш → Отиваш на регистрация
- Ако съществуваш + парола OK + платил си → Чат
- Ако съществуваш + НЕ си платил → Плащане

### **Регистрация:**
1. Телефон (вече въведен)
2. Парола (вече въведена)
3. Име
4. Пол (male/female)
5. Височина (optional)
6. Тегло (optional)
7. Държава (optional)
8. → Плащане (€5 или $5)

---

## 💳 Плащане

**Цена:** €5/месец (ЕС) или $5/месец (извън ЕС)

**Stripe Checkout:**
1. Автоматично redirect към Stripe
2. Въведи карта
3. Плати
4. Redirect обратно → Чат

**Test Cards:**
- Success: `4242 4242 4242 4242`
- Declined: `4000 0000 0000 0002`

---

## 💬 Чат

### **Добавяне на контакт:**

**Опция 1: По телефон (познат)**
1. Меню → "Добави контакт"
2. Въведи телефонен номер
3. Ако съществува → Добавя се
4. Започваш чат

**Опция 2: По критерии (запознанство)**
1. Меню → "Търси хора"
2. Филтри:
   - Държава
   - Пол (male/female)
   - Височина (exact, напр. 180cm)
   - Тегло (exact, напр. 75kg)
3. Избираш от резултатите
4. Добавяш като контакт
5. Започваш чат

**❌ НЕ можеш да търсиш по:**
- Име
- Град
- Улица
- Работно място

---

### **Изпращане на съобщения:**

**Текст:**
1. Пишеш в полето
2. Enter или бутон "Изпрати"

**Файлове (до 100MB):**
1. Бутон "📎 Attach"
2. Избираш файл
3. Upload → Изпраща се
4. Другият сваля → **Файлът се трие автоматично!**

**Местоположение:**
1. Бутон "📍 Location"
2. Browser/App иска разрешение
3. Позволяваш → Изпраща:
   - Държава, град, село, улица, номер
   - GPS координати
   - IP адрес
   - Линкове: Google Maps, 2GIS, Yandex

---

### **История:**

- Само последните **5KB** текст (~100 съобщения)
- Старото **се трие автоматично**
- Файлове се трият след **сваляне** или **24 часа**

---

## 🔒 Анонимност

### ✅ **Какво е анонимно:**
- Без SMS верификация
- Без email
- Без Facebook/Google login
- История се трие (5KB limit)
- Файлове се трият

### ❌ **Какво НЕ Е анонимно:**
- Телефонният номер е видим за контактите
- Плащането е чрез Stripe (има payment data)
- Admin може да те блокира
- Критични думи се мониторират

---

## 🚫 Блокиране

Ако използваш критични думи (drugs, weapon, terror, kill, etc.):
- Разговорът се маркира
- Admin го проверява
- Admin може да те **блокира**

**Блокиран акаунт:**
- Не можеш да влезеш
- Контактите НЕ виждат съобщенията ти
