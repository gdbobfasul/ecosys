# Pupikes Toolkit Price Watch (Price-Watch Bot) — как функционира

Самостоятелно мобилно приложение „робот под наем", което следи **цени на криптовалути и валути** и праща **локални** известия при достигнат праг. Без акаунт, без облак, без проследяване. Всички цени идват от **безплатни публични API без ключове**. Без крипто плащания — само информация за цени. Издава се за **RUStore** и **Huawei AppGallery** като две идентични, независими приложения.

- **App ID:** `com.pupikes.pricewatchbot.rustore` (RUStore) / `com.pupikes.pricewatchbot.huawei` (Huawei)
- **Стек:** Vanilla JS + Vite 5 + Capacitor 6

## Поток (наеми → настрой → разреши → следи)
1. **Наеми робота (онбординг)** — обяснение + бутон „Активирай безплатно" (премиум наемът е демонстрационен STUB).
2. **Настрой робота** — добавяш наблюдения: крипто или валута (FX), условие „под/над", прагова стойност, честота (15 мин / 1 ч / дневно).
3. **Разрешения** — иска само Известия (по желание) + Интернет; нищо излишно.
4. **Табло** — главен ключ ON/OFF, списък наблюдения с последна стойност и статус, дневник, пауза/изтриване, „Провери сега".

## Безплатни източници (без ключ, без акаунт)
- Крипто (основен): Binance — `api.binance.com/.../ticker/price`
- Крипто (резервен): CoinGecko — `api.coingecko.com/.../simple/price`
- Валути: open.er-api — `open.er-api.com/v6/latest/USD`

Нито един адрес не изисква API ключ или регистрация.

## Известия и планиране — уеб срещу устройство
- **Устройство:** известията минават през `@capacitor/local-notifications` (локални, без сървър/push). Планировчикът работи, докато приложението е на преден план.
- **Уеб/preview:** планировчикът ползва `setInterval`; известията са Web Notifications.
- **Истински фонов режим** (угаснал екран) изисква foreground service или background-runner плъгин. Скелетът е в `src/core/scheduler.js` (`registerBackgroundRunner`) и е маркиран като TODO — честно документирано.

## Управление на данните
- Конфигурацията на наблюденията (символ, условие, праг, честота) и последните стойности се пазят **само локално** през `@capacitor/preferences`.
- Няма акаунт, няма облак; всяко наблюдение и цялата конфигурация се трият от потребителя.

## Разрешения
- **INTERNET** — за теглене на котировки от изброените keyless публични API (само заявки за четене на цени).
- **POST_NOTIFICATIONS** — локални известия при преминаване на праг (незадължително).

## Езици
Интерфейсът е на **15-те езика на екосистемата** (bg, ru, uk, en, de, fr, es, es-MX, it, pt, ar, hi, ja, ky, zh-Hant). Език по подразбиране: **руски**.
