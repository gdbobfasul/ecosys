# Stay Informed (RUStore издание)

Самостоятелно мобилно приложение за новини по държави от целия свят.

- **Обхват:** всичките ~195 държави. За всяка има национална агрегаторна емисия (Google News
  на местния език); за водещите държави — и поименни официални/неофициални източници
  (вестник/телевизия/радио/агенция) през RSS. Виж `src/data/feeds.js`.
- **Език:** 15-те езика на екосистемата, дефолт руски; избор при старт и по всяко време с 🌐.
- **Превод:** заглавията се превеждат на избрания език наживо през MyMemory (без ключ, с кеш).
- **Четене на глас:** браузърен `speechSynthesis` (без външен плъгин).
- **Без сървър, без акаунти, без проследяване, без крипто.** Всичко тече на телефона;
  заявките минават през CapacitorHttp (заобикаля CORS).

## Билд
```
deploy-scripts/build-mobile-apps.sh rustore/stay-informed
```

Източниците принадлежат на своите собственици; приложението е независим четец.
