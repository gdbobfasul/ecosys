# Тестване — Рутина-робот (AppGallery)

## В браузър (бързо, без Android SDK)
1. `npm install`
2. `npm run dev` → отвори показания URL (обикновено http://localhost:5173).
3. Поток: Онбординг → „Активирай робота" → конфигурирай рутина → напомняния → разрешения → табло.
4. „Преглед на брифинга сега" показва известие (Web Notifications, ако са разрешени, иначе вграден toast).
5. Задади близък час на напомняне (1–2 мин напред) и остави таба отворен → известието идва (уеб таймер).

## Поведение на разрешенията
- Известия: бутонът иска уеб/нативно разрешение.
- Локация: пробвай „Използвай локацията" → ако откажеш, въведи град „София" → координати се попълват → брифингът показва времето.

## На устройство (Android)
1. `npm run build`
2. `npm run cap:add` (еднократно), `npm run cap:sync`
3. `npm run cap:open` → стартирай от Android Studio на емулатор/устройство.
4. Нативните локални известия се планират чрез `@capacitor/local-notifications` и идват и при затворено приложение (повтарящи се по час).

## Известно ограничение (уеб)
В браузър планирането използва `setTimeout` и работи само докато табът е отворен. На устройство се ползват нативни локални известия.

## Граничен / офлайн
- Без мрежа: брифингът показва „времето: няма връзка", останалото работи.
- Без зададена локация: подканя да зададеш локация.
