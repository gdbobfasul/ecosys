# Testing — Auto-Reply Bot (Huawei AppGallery)

## В браузъра (без Android SDK)
1. `npm install`
2. `npm run dev` → отвори показания URL (по подразбиране http://localhost:5182).
3. На онбординга натисни **„Наеми / Активирай робота"**.
4. В **Правила**: добави правило (тригер „съдържа" → дума `цена`, отговор по избор). Запази график 24/7.
5. В **Табло**: включи робота (ON).
6. В **Demo Inbox**: като подател „Иван" напиши „Каква е цената?" → роботът отговаря; в **Табло → Дневник** се появява запис; ако са разрешени известия, идва toast/нотификация.

## Сценарии за проверка
- **Тригери:** contains / exact / any реагират според очакванията.
- **Приоритет:** при няколко съвпадащи правила печели по-горното (ред = приоритет).
- **Office-hours:** режим „Работно време" извън прозореца → връща away-съобщението; вътре → нормални правила. Тествай и прозорец през полунощ (напр. 22:00–06:00).
- **Списъци:** черен списък игнорира подателя; непразен бял списък допуска само изброените.
- **Pause/Delete/Reorder:** правилата се паузират, трият и местят.
- **Известия:** при разрешение идва локално известие; без него — in-page toast.

## На устройство (изисква Android SDK/JDK — извън тази среда)
1. `npm run build`
2. `npx cap add android` (еднократно), после `npx cap sync android`.
3. Отвори `android/` в Android Studio, билдни APK/AAB. (Тук НЕ се прави — няма SDK/gradle.)
