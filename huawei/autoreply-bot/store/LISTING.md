# Pupikes Auto Answer — Store Listing (Huawei AppGallery)

**App ID:** `com.kcy.autoreplybot.huawei`
**Category:** Productivity / Utilities

## Short description
Наеми робот, който отговаря автоматично на съобщения по твои правила — изцяло на устройството.

## Full description
Pupikes Auto Answer е „робот под наем", който отговаря вместо теб по предварително зададени
правила: ключови думи, готови шаблони с променливи ({name}, {time}, {date}), работно
време (office-hours) и бели/черни списъци по подател.

Поради ограниченията на Android (системните SMS и чужди приложения НЕ могат да се
автоматизират от трета страна), роботът работи в **собствен симулиран чат (Demo Inbox)**
вътре в приложението. Там виждаш и тестваш точно как би отговарял.

### Възможности
- Активиране/„наемане" на робот (безплатно ядро).
- Конфигуратор на правила с приоритети, пауза, пренареждане.
- Тригери: съдържа дума / точно съвпадение / всяко съобщение.
- Шаблони с променливи и автоматичен away-отговор извън работно време.
- Бели/черни списъци по име на подател.
- Локални известия при изпратен авто-отговор.
- Дневник на всички авто-отговори.

### Поверителност
100% на устройството. Без интернет, без акаунти, без контакти, без достъп до SMS,
без проследяване. Виж `store/DATA-SAFETY.md`.

### Покупки в приложението
„Premium наем" е по избор и в този билд е STUB (симулиран, без реално плащане).
