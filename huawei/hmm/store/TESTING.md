# Тестване — Битка на терен (HMM), Huawei AppGallery

## Бърз тест в браузър (без Android)

```bash
npm install
npm run dev
```

Отвори показания URL (по подразбиране http://localhost:5174).

### Контролен лист на геймплея

- [ ] Менюто се показва със заглавие и бутон „Започни".
- [ ] Полето се рисува с жив терен (движещи се листа/животни отстрани на пътя).
- [ ] На всяко ниво излизат 3 съюзника (вляво) и 3 врага (вдясно).
- [ ] Героите имат видео-анимации: idle, придвижване, удар, реакция при щета.
- [ ] Бутоните V/B нанасят обикновени удари; HP баровете намаляват.
- [ ] Комбо-бутоните (6 на герой) откриват специален удар при познати 4.
- [ ] Откритият спец се пуска със зеления бутон „⚡ Спец".
- [ ] Стрелките/тапването сменят целта при 2+ живи врага.
- [ ] При победа на всички врагове → екран „ниво преминато" → следващо ниво.
- [ ] При смърт на целия отряд → екран „Загуба".
- [ ] Рекордът се запазва (презареди — рекордът остава; localStorage).

### Офлайн проверка

- [ ] Изключи мрежата (DevTools → Network → Offline) и презареди — играта пак тръгва.
- [ ] В Network таб НЯМА заявки към `/api/portals/...` или `fonts.googleapis.com`.

## Тест на устройство (след `cap sync`, изисква Android SDK)

- [ ] Стартирай на телефон/емулатор; играта се вижда на цял екран.
- [ ] Всички ходове са изпълними само с пръст (без хардуерна клавиатура).
- [ ] Завъртане/мащабиране — полето остава центрирано (letterbox).
