# Тестване — Цена-робот (Huawei AppGallery)

## В браузър (без Android SDK)
1. `npm install`
2. `npm run dev` и отвори показания URL (обикновено http://localhost:5173).
3. Поток: „Активирай робота" → добави наблюдение (напр. BTC „качи се над 1") →
   „Продължи към разрешения" → разреши известия → на таблото пусни главния ключ.
4. Натисни „Провери сега" — стойността се обновява от Binance; ако условието е
   изпълнено, се появява уеб известие и запис в дневника.
5. Тест на резервния източник: при недостъпен Binance заявката пада към CoinGecko.
6. Тест на FX: добави EUR „качи се над 0.1" и „Провери сега".

## Офлайн/грешки
- Изключи мрежата и натисни „Провери сега" — наблюдението показва статус „грешка"
  и запис в дневника, без приложението да блокира.

## На устройство (когато има среда с Android SDK/JDK)
1. `npm run build`
2. `npm run cap:add` (еднократно), после `npm run cap:sync`.
3. Отвори в Android Studio (`npm run cap:open`) и пусни на емулатор/телефон.
4. Провери, че известията са системни (locale notifications), не уеб.
5. Без HMS Core — известията са локални, не push.

## Демо-покупка (IAP STUB)
- „Наеми премиум (демо)" вдига лимита на наблюденията. Не извършва реално плащане
  (виж `src/store/huawei-sdk.js`).
