# Huawei AppGallery — Тестова среда / Закрито тестване

## 1. Браузър (основна тестова среда за разработка)
```
npm install
npm run dev      # отваря на http://localhost:5173
```
Играй директно в браузъра — това е средата за бърз тест на геймплея.

Управление (клавиатура):
- A / D или стрелки — движение
- W — скок
- SPACE — атака, SHIFT+SPACE — специален удар
- 1–5 — смяна на оръжие (само отключените)

На телефон: екранни бутони (D-pad ляво, ⚔/★/⤒ дясно).

## 2. Закрито тестване в AppGallery Connect
AppGallery Connect -> My apps -> приложението -> "Operate" / "Testing":
1. Качи подписан APK/AAB (debug за вътрешен тест; release подпис за публикация).
2. Създай "Closed test" / "Open test" група и добави тестери (имейли/HUAWEI ID).
3. Тестерите получават линк за инсталация през AppGallery.
4. Тъй като играта няма IAP/мрежа, не е нужна sandbox среда за плащания.

Бележка: AppGallery изисква **64-bit** билд — Capacitor/Gradle произвежда arm64-v8a по
подразбиране, така че това условие е покрито.

## 3. Подписване
- Генерирай keystore (`keytool`), конфигурирай в `android/app/build.gradle` (release).
- В AppGallery Connect може да ползваш и "App Signing" (управлявани ключове).
