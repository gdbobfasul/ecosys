# Huawei AppGallery — затворено тестване / тестова среда

## Тестова среда (без магазин)
Основната тестова среда е **браузърът**:
```bash
npm install && npm run dev
```
Всеки офлайн инструмент е напълно функционален в браузъра. APK за устройство се
билдва отделно (виж README → Android APK; локалната машина няма Android SDK).

## Затворено тестване в AppGallery
Huawei AppGallery Connect поддържа:
1. **Open/Closed testing (Тестване)** — качваш APK/AAB и каниш тестери чрез групи
   с e-mail адреси; те получават линк за инсталиране.
2. **Draft / вътрешна версия** — преди публична модерация.

### Стъпки
1. Регистрирай се в AppGallery Connect: https://developer.huawei.com/consumer/
   (изисква верификация на акаунт за разработчик).
2. Създай ново приложение → Package name `com.kcy.servicestoolkit.huawei`.
3. Качи подписан APK/AAB (debug е достатъчен за вътрешен тест; за публикуване —
   release с keystore — виж submission-checklist.md).
4. Попълни листинга (listing-notes.md), икона, екранни снимки и privacy policy URL.
5. Раздел „Testing" → добави тестова група и тестери по e-mail → разпрати линка.
6. След потвърждение → подай за официална проверка (review).

## Контролен лист за тестера (ръчно на устройство)
- [ ] Началният екран показва всички карти; търсачката филтрира.
- [ ] QR: генерира код, сваля PNG; разчита от снимка; сканира с камера.
- [ ] Парола: четирите метода работят; индикаторът за сила; копиране.
- [ ] Калкулатор: 4-те таба дават коректни числа.
- [ ] Текст: броячите се обновяват; форматиранията и Base64 работят.
- [ ] Снимки: компресира и сваля; показва спестения размер.
- [ ] PDF: слива, разделя, добавя воден знак; визуален подпис се поставя.
- [ ] Свиване на PDF: намалява размера и сваля файла.
- [ ] Онлайн екраните показват ясна бележка „изисква онлайн услуга".
- [ ] Приложението работи в самолетен режим (офлайн инструментите).
- [ ] Работи на Huawei устройство БЕЗ Google услуги (HMS-only телефон).
