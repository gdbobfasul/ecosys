# Huntline 3D — Huawei AppGallery Listing Notes

App ID: `com.pupikes.fpshunter.huawei`
Name: Huntline 3D
Category: Games / Action (Shooter)

## Кратко описание (BG)
3D ловен шутър от първо лице. 100 процедурно генерирани нива през гори, полета,
сняг, пустиня, хълмове, блата и градски руини. Лови различни цели — сърни, елени,
зайци, змии, гноми, войници, танкове, самолети — с подходящото за тях оръжие.

## Short description (EN, за AppGallery аудитория)
3D first-person hunting shooter. 100 procedurally generated levels across forests,
fields, snow, desert, hills, marsh and urban ruins. Hunt different targets with the
right weapon for each.

## Пълно описание
- 100 нива с нарастваща трудност (повече цели, по-бързи и уклончиви, по-малко време/боеприпаси).
- Всяко ниво има ЕДИН тип цел; оръжието се избира автоматично спрямо целта
  (пушка за дивеч, пистолет за змии, прашка за гноми, ракети за танкове/самолети).
- 3D от първо лице: телефон — ляв джойстик + влачене за оглеждане + бутон ОГЪН;
  десктоп — WASD + мишка (pointer lock) + клик за стрелба.
- Процедурен терен и нискополигонална стилизирана графика, генерирана изцяло в код.
- Локален лидерборд (Топ-10) — пишеш само име, точките се класират на устройството.
- Изцяло офлайн. Без реклами. Без вход. Без мрежа.

## Технически
- HTML5 (Three.js за 3D) опакован с Capacitor 6.
- Локален лидерборд чрез SQLite (sql.js / WASM), персистиран в IndexedDB.
- Без Huawei Mobile Services (HMS) / Google Mobile Services / Firebase.
- Без интернет разрешения.
- minSdk 23 / target 34 (по подразбиране на Capacitor 6).

## Поверителност / Data Safety
Приложението съхранява САМО локален лидерборд: свободно въведено ИМЕ + ТОЧКИ.
Без акаунти, без контакти, без телефонни номера, без идентификатори, без аналитика,
без мрежа. Данните не напускат устройството и могат да се изтрият с деинсталиране.

## Възрастов рейтинг
12+ (стилизирана стрелба по цели, без кръв/реализъм).
