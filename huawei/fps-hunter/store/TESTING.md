# Huawei AppGallery — Тестова среда / Закрито тестване (Huntline 3D)

## 1. Браузър (основна тестова среда за разработка)
```
npm install
npm run dev      # отваря на http://localhost:5173
```
Играй директно в браузъра — това е средата за бърз тест на геймплея.

Управление:
- Десктоп: WASD движение, мишка за оглеждане (кликни веднъж за заключване на курсора),
  ляв бутон на мишката = стрелба.
- Телефон/тъч: ляв джойстик (влачи в лявата половина) за движение, влачи в дясната
  половина за оглеждане, бутон ОГЪН долу вдясно.

В менюто избираш начално ниво (1–100) за бърз тест на конкретен биом/цел/оръжие.

## 2. Закрито тестване в AppGallery (Closed/Internal testing)
AppGallery Connect -> My apps -> приложението -> "Testing":
1. Качи подписан APK/AAB (debug е достатъчен за вътрешен тест; за публикация — release подпис).
2. Добави тестери (Huawei ID / имейли) в групата за тестване.
3. Тестерите инсталират през линка за тестване.
4. Понеже играта няма билинг/мрежа, не е нужна sandbox среда за плащания (HMS IAP).

## 3. Подписване
- Генерирай keystore (`keytool`), конфигурирай в `android/app/build.gradle` (release).
- Качи signing информацията в AppGallery Connect (App signing).
