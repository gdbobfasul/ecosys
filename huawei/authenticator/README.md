# KCY Authenticator (RUStore)

Двуфакторен автентикатор (като Google Authenticator / Aegis), напълно офлайн.

## Възможности
- **TOTP** (RFC 6238), **HOTP** (RFC 4226) и **Steam Guard** кодове.
- Добавяне на акаунт чрез **сканиране на QR код** (камера + jsQR) или **ръчно** (вкл. поставяне на `otpauth://` линк).
- **Шифриран сейф**: тайните се пазят AES-GCM, ключът е изведен от master парола през PBKDF2 (210 000 итерации, SHA-256). Без паролата няма достъп.
- **Биометрично отключване** (пръстов отпечатък/лице) — паролата се пази в Android Keystore през защитеното хранилище. Деградира тихо до парола, ако устройството не поддържа.
- **Авто-заключване** при бездействие и при отиване на заден план.
- Търсене, редакция, подредба, изтриване.
- **Импорт/Експорт** на акаунтите (`.json`).
- Интерфейс на **15 езика** (език по подразбиране: руски), с екран за избор на език при първо стартиране.

## Архитектура
- `src/core/base32.js` — Base32 (RFC 4648).
- `src/core/otp.js` — TOTP/HOTP/Steam + разбор/сглобяване на `otpauth://`.
- `src/core/vault.js` — AES-GCM + PBKDF2 шифроване на сейфа.
- `src/core/storage.js` — съхранение + сесийно състояние + операции върху записите.
- `src/core/biometric.js` — биометрия (предпазливо зареждане на Capacitor плъгините).
- `src/core/i18n.js` + `languages.js` — преводи на 15-те езика.
- `src/screens/*` — екрани (език, създаване, отключване, списък, добавяне, редакция, настройки).

## Разработка
```
npm install
npm run dev
```

## Билд (Android / Capacitor)
```
npm run build
npx cap sync
npx cap open android
```

Сигурност: тайните никога не напускат устройството и не се пазят в чист вид (освен в ръчно поискан експорт-файл, който предупреждаваме да се пази на сигурно).
