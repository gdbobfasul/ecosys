# Рустам (Huawei AppGallery)

Самостоятелна аркадна игра на Phaser 3 + Capacitor. Рустам стои насред поле и бере
краставици, които изскачат от земята. До всяка краставица започва да расте **дупка** —
от малка точица постепенно се уголемява. Стигне ли пълен размер, от нея изскача
**къртица**, която прибира краставицата под земята (пропусната). Рустам трябва да стигне
краставицата **навреме**.

## Правила
- Движиш Рустам с виртуалния джойстик (докосни и влачи където и да е), или WASD/стрелки на десктоп.
- Бери по-бързо от къртиците: набери **целта** краставици, за да минеш нивото.
- Всяка краставица, която къртица отмъкне, ти коства едно сърце (пропуск). Свършат ли сърцата — край.
- **10 нива.** След 3-то ниво къртиците (и дупките) стават с 1-2 пиксела по-малки на всяко ниво,
  а времето на Рустам до краставицата намалява. Виж `src/scenes/levels.js`.

## Технология
- Phaser 3 (`phaser`), Vite, Capacitor 6 (Android). Без Google Mobile Services.
- Всички текстури са **процедурни** (рисуват се в кода — `src/gfx/textures.js`), без външни картинки.
- Ранг листа и прогрес — само локално в `localStorage` (без сървър, без контакти, без мрежа).

## Стартиране
```bash
npm install
npm run dev        # http://localhost:5177
npm run build      # → dist/
npm run cap:add    # еднократно: добавя android платформата
npm run cap:sync   # build + sync към android
```

## Структура
| Файл | Роля |
|------|------|
| `src/main.js` | Phaser конфиг + резолюция, списък със сцените |
| `src/theme.js` | палитра/заглавия/ключ за записа (различен от Huawei билда) |
| `src/scenes/levels.js` | данните за 10-те нива (цел, сърца, размер на къртица/дупка, време) |
| `src/scenes/game.js` | полето, Рустам, изскачането на краставиците, бране/пропуск, победа/загуба |
| `src/scenes/ui.js` | HUD (сърца, прогрес, точки) + виртуален джойстик |
| `src/scenes/menu.js` | меню, избор на ниво, ранг листа |
| `src/scenes/gameover.js` | край + въвеждане на име + ранг листа |
| `src/entities/rustam.js` | героят (движение) |
| `src/entities/cucumber.js` | краставица + дупка + къртица (жизнен цикъл) |
| `src/gfx/textures.js` | всички процедурни текстури |
