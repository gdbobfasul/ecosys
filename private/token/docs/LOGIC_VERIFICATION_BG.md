## 🔍 ПРОВЕРКА НА КОДА

### При Deployment (Constructor):

```solidity
balanceOf[DEV_WALLET_mm_vis] = 600_000 * 10**decimals;  // 600k резервирани
```

**Резултат след deploy:**
```
┌─────────────────────────────────────────┐
│ DEV_WALLET_mm_vis: 600,000 токена       │ ← РЕЗЕРВИРАНИ за exempt портфейли
│ Contract: 400,000 токена                │ ← За ПРОДАЖБА (ликвидност, купуване)

```

### При Натискане на "Distribute" Бутон:


**Резултат след distribute:**
```

│ РЕЗЕРВИРАНИ (600k) се разпределят така: │

┌─────────────────────────────────────────┐
│ ЗА ПРОДАЖБА (400k) остават така:        │
├─────────────────────────────────────────┤
│ Contract: 400,000                       │ ← НЕ се променят!
├─────────────────────────────────────────┤
│ ЗА ПРОДАЖБА: 400,000 ✅                 │
```

## 🎯 Ключови Факти

1. **600,000 = РЕЗЕРВИРАНИ**
   - За exempt портфейли (team, marketing, advisor, dev)
   - Разпределят се при натискане на distribute бутон
   - НЕ са за продажба на обикновени хора

2. **400,000 = ЗА ПРОДАЖБА**
   - Остават в contract
   - НЕ се разпределят към exempt портфейли
   - От тях хората купуват токени
   - Използват се за ликвидност в DEX

3. **Distribution Бутон**
   - Изпраща 500k от DEV_WALLET_mm_vis към другите
   - 100k остават в DEV_WALLET_mm_vis
   - Contract токените (400k) НЕ се променят
